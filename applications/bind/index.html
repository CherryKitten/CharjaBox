BIND

BIND - Domain Name Server for your home network

Please note that BIND requires some additional setup after installing. For reference you can use the second half of this tutorial and the BIND documentation.

The docker container used for BIND comes with an instance of Webmin to make management of your DNS easier. You can disable this by setting bind_webmin_enabled to "false".

BIND has to run on port 53 for DNS to work. The Webinterface port can be set by changing bind_webmin_port.
