Nginx

Nginx - Open source web server and a reverse proxy server

nginx_config_directory binds to /etc/nginx, for information on how to configure nginx see the nginx documentation.

nginx_data_directory binds to /var/www/, put everything the webserver should serve in here.

For information on how to setup nginx as a reverse proxy for your other apps, see this documentation page.
